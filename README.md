# Xibw Bot

Xibw Botは、X（旧Twitter）APIと連携したDiscord Botです。Xのアカウントやハッシュタグのデータをリアルタイムで取得し、Discordに通知する機能を提供します。また、YouTube、Instagram、Twitchのリンクにも対応し、関連情報を自動で取得して投稿します。

[![Discord](https://img.shields.io/badge/%20discord-blue.svg)](https://discord.gg/bfR9GNGxgM)

## **Xibw Bot バージョン一覧**  
### V1 Twitter連携(/x)
- user : 特定人物の投稿を送信
- url : URLの投稿を送信・x.comのURLを張ると自動で送信
- hash : hashの投稿を送信
- poll : 最新の投稿を送信（再読み込み）
- adult : アダルトコンテンツの停止/開始
など、DiscordBotのコマンドを見てください
### V2 Karotter連携(/Karotter)
- user : 特定人物の投稿を送信
- url : URLの投稿を送信・karotter.comのURLを張ると自動で送信
- hash : hashの投稿を送信
- poll : 最新の投稿を送信（再読み込み）
- adult : アダルトコンテンツの停止/開始
など、DiscordBotのコマンドを見てください
